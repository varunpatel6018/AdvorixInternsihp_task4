# AdvorixInternsihp_task4
Pyhton Programming Project ( Task 4) Data visualisation 

📊 Student Data Visualization using Matplotlib
📌 Project Description

This project is a simple Data Visualization program developed using Python.
It visualizes student academic data (such as marks in different subjects) using various graphical representations like bar charts, line graphs, and pie charts.

The main objective of this project is to understand how data can be visually represented using the Matplotlib library.

🎯 Objectives

To learn the basics of data visualization

To understand how to use Matplotlib in Python

To represent student marks in graphical form

To compare performance using charts

🛠 Technologies Used

 Python 3

📊 Matplotlib Library

📚 Libraries Used
import matplotlib.pyplot as plt

(NumPy may be used optionally if required for numerical positioning.)

📈 Features of the Project

Displays student marks using:

Bar Chart

Line Graph

Pie Chart

Easy to understand and modify

Beginner-friendly code

Clean and simple structure

▶ How to Run the Project

Install Python (if not already installed)

Install Matplotlib using:

pip install matplotlib

Open terminal in project folder

Run the program:

python AdvorixInternship_task4.py
